COMET GAME STORE

Project information

  • Category: Web design
  • Engine: HTML/CSS
  • Project date: May 2020

A website built for selling games, inspired by other famous internet game-stores. This is made to show the power of HTML/CSS and Bootstrap library in making front-end.
Bonus: You can view the Youtube trailer of each game shown in this website, by clicking on the game card/carousel's picture.
