Smith's Games Beacon

Project information

  • Category: Web app
  • Frontend Engine: ReactJS/React-router/Redux-thunk
  • Backend Engine: NodeJS (Express)
  • Database: MongoDB
  • Project date: July 2020

An App to search for your favorite PC video games with the best sale price for it, in order to solve piracy issues rampant in the PC game industry. You can search games by popularity, rating, genres, tags. The app uses different APIs to search for the game's detail and its price, and stores the user data in the app's backend.
Bonus: You can wishlist any game you want and see it on your Profile page!
Bonus 2: You can also put the game into cart and "buy" it. The checkout functionality is not completed yet, but you can see also the game purchased in your Profile and your cart emptied.
