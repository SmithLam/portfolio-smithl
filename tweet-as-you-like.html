Tweet as You Like!

Project information

  • Category: Web design
  • Engine: HTML/CSS/JS
  • Project date: June 2020

A website built after Twitter, where you can tweet anything you like. You can also delete, retweet, hashtag and filter via hashtag.
Bonus: You can post images via url (.png, .jpg, .gif are supported) and you can watch Youtube video, also by posting the url. So the website can serve as a small media center/Youtube station.
